# 小丑牌 Web V1.0.0

基于扑克牌型的单机数值小游戏。手选 1–5 张组牌型，限 4 次出牌内累计超过 300 分即胜。

## 玩法

- 每局随机发 8 张手牌（标准 52 张牌堆）
- 点击手牌选中（橙色高亮+上浮），再点取消，最多选 5 张
- **出牌**：识别牌型计分，累计分数；**弃牌**：换牌不得分
- 出牌 4 次内累计 ≥ 300 分胜利，否则失败

## 牌型与计分

| 牌型 | 基础分 | 倍数 |
|------|--------|------|
| 高牌 | 5 | ×1 |
| 对子 | 10 | ×2 |
| 两对 | 20 | ×2 |
| 三条 | 30 | ×3 |
| 顺子 | 30 | ×4 |
| 同花 | 35 | ×4 |
| 葫芦 | 40 | ×4 |
| 四条 | 60 | ×7 |
| 同花顺 | 100 | ×8 |
| 皇家同花顺 | 100 | ×8 |

得分 = (基础分 + 出牌点数之和) × 倍数

## 使用方式

无需安装，直接用浏览器打开 `index.html` 即可游玩。
